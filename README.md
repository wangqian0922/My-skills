# My Skills

这是一个用于管理个人 Skills 的仓库。

## 目标

- 将每个 Skill 作为一个独立单元进行管理
- 通过统一目录结构，便于新增、维护和复用

## 存储规范

- 一个文件夹对应一个独立 Skill
- Skill 名称建议使用清晰、可读的英文或拼音
- 每个 Skill 文件夹内至少包含一个说明文件（例如 `README.md` 或 `SKILL.md`）

## 建议目录结构

```text
.
├─ README.md
├─ skill-a/
│  └─ SKILL.md
├─ skill-b/
│  └─ SKILL.md
└─ skill-c/
   └─ SKILL.md
```

## 使用建议

- 新建 Skill 时，先创建对应文件夹，再补充该 Skill 的用途、输入、输出和示例
- 变更 Skill 内容时，保持提交信息清晰，便于追踪历史
